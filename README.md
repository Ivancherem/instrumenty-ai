# 📚 Instrumenty-AI — Образовательный портал

**Instrumenty-AI** — это современный образовательный портал с бесплатными инструментами для учебы, курсами, блогом и ИИ-помощником.

🔗 **Сайт:** [https://instrumenty-ai.ru](https://instrumenty-ai.ru)

---

## ✨ Возможности

### 🛠️ Инструменты
- **📝 Генератор тестов** — создавайте тесты, проходите их и получайте результаты
- **🤖 ИИ-помощник** — задавайте вопросы об учебе, программировании, английском
- **📚 Конструктор уроков** — создавайте структурированные уроки
- **📊 Анализ успеваемости** — отслеживайте оценки и прогресс

### 📰 Блог
- 6 статей об образовании, ИИ и эффективном обучении

### 🎓 Курсы
- 6 бесплатных курсов по разным направлениям
- Запись на курсы и отслеживание прогресса
- Личный кабинет "Мои курсы"

### 📄 Страницы
- Главная, О проекте, Контакты
